# Project Authoring Helpers

## Purpose

Own project-local, imported helpers used to author and maintain skills in this repository.

## Ownership

- `.agents/skills/write-a-skill/` is an installed authoring aid, not a published skill package from this repository.
- Imported helper content remains distinct from first-party packages under `skills/`.

## Local Contracts

- Treat installed helper files as external material; update them through their source/install workflow rather than incidental feature edits.
- Do not list this subtree in a first-party skill manifest.

## Work Guidance

- Read the applicable helper instructions when using an imported authoring workflow.

## Verification

- Confirm first-party package changes do not accidentally modify imported helper files.
